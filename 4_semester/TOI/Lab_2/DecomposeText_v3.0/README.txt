TO INSTALL

(1) copy DecomposeText.jsx into the following folder of your After Effects
installation:

"Support Files/Scripts/ScriptUI Panels" (on Windows)
"Scripts/ScriptUI Panels" (on Mac) 

(2) Start DecomposeText (via the "Window" menu in After Effects)

List of changes in v3:

. 2 options: keep original position using expressions (but new layers have the same text as the original layer with some elements hidden), approximated position without expressions (but new layers are 1 char/word/line long)
. small bugfix in the decomposition
. throw alert when paragraph text is detected (not supported)
. rewrite Help/About box
. AE CC compatibility
